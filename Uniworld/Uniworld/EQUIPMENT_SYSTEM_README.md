# Система экипировки персонажа

## Описание
Добавлена полнофункциональная система экипировки персонажа для вашей визуальной новеллы на Ren'Py.

## Возможности
- ✅ Визуальное отображение персонажа с экипировкой
- ✅ 6 слотов экипировки: нижнее белье, аксессуары, штаны, торс, головные уборы, оружие
- ✅ Инвентарь с сеткой предметов
- ✅ Клик для экипировки/снятия предметов
- ✅ Система характеристик (сила, ловкость, интеллект, харизма)
- ✅ Автоматическое сохранение состояния экипировки

## Как использовать
1. В игре выберите "Экипировка персонажа" в главном меню
2. Кликайте по предметам в инвентаре справа, чтобы экипировать их
3. Кликайте по экипированным предметам на персонаже, чтобы снять их
4. Характеристики персонажа отображаются внизу слева

## Структура файлов изображений
Поместите PNG изображения в следующие папки:

```
game/images/
├── character/
│   └── base_character.png          # Базовое тело персонажа 640x960
├── equipment/
│   ├── underwear/
│   │   ├── basic_underwear.png
│   │   └── silk_underwear.png
│   ├── accessory/
│   │   ├── silver_ring.png
│   │   └── gold_necklace.png
│   ├── pants/
│   │   ├── cloth_pants.png
│   │   └── leather_pants.png
│   ├── torso/
│   │   ├── cloth_shirt.png
│   │   └── leather_armor.png
│   ├── hat/
│   │   ├── cloth_cap.png
│   │   └── iron_helmet.png
│   └── weapon/
│       ├── wooden_sword.png
│       └── iron_sword.png
```

## Требования к изображениям
- Базовое тело персонажа: 640x960 пикселей
- Экипировка: PNG с прозрачным фоном, размер 640x960 пикселей
- Все изображения экипировки должны совпадать по размеру с базовым телом

## Добавление новых предметов
Чтобы добавить новые предметы экипировки:

1. Добавьте определение предмета в `equipment_data` в script.rpy
2. Добавьте ID предмета в соответствующий список в `inventory_items`
3. Поместите PNG файл в соответствующую папку

Пример добавления нового предмета:
```python
"new_armor": {
    "id": "new_armor",
    "name": "Новая броня",
    "image": "equipment/torso/new_armor.png",
    "slot": "torso",
    "stats": {"strength": 5, "agility": 0, "intelligence": 0, "charisma": 2},
    "description": "Мощная новая броня"
}
```

## Характеристики предметов
Каждый предмет может изменять 4 характеристики:
- `strength` - Сила
- `agility` - Ловкость  
- `intelligence` - Интеллект
- `charisma` - Харизма

Значения могут быть положительными (бонус) или отрицательными (штраф).

## Техническая информация
- Все данные автоматически сохраняются в save-файлах игры
- Система обрабатывает отсутствующие изображения без сбоев
- Код полностью интегрирован в существующую структуру игры
